Hi everyone, I'm **Manh Huynh**.

**Freelance Flow** is a project I built to help freelancers and small teams manage their work more effectively. I wanted to create a tool that is simple, intuitive, and gets out of your way so you can focus on what matters.

It's built with modern web technologies and uses Supabase for secure data storage.

### ‚ú® What can it do?

*   **Organize Tasks**: View your work your way‚ÄîCalendar, Kanban, List, or Timeline.
*   **Manage Clients**: Keep track of client details and project history.
*   **Create Quotes**: Generate professional quotes with timelines and export them easily.
*   **AI Assistance**: I've integrated Google Gemini to help with things like drafting tasks or analyzing project data.
*   **Secure & Private**: Your data is stored securely in your own Supabase account.

### üõ†Ô∏è Tech Stack

*   **Framework**: Next.js 14, React, TypeScript
*   **Styling**: TailwindCSS, shadcn/ui
*   **Database**: Supabase (PostgreSQL)
*   **AI**: Google Gemini
*   **Deployment**: Vercel (recommended)

### üöÄ Setup for Developers

If you want to run this project locally or contribute:

1.  **Clone the repo**
    ```bash
    git clone https://github.com/manhhuynh-designer/Freelance-Flow.git
    cd Freelance-Flow
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Environment Setup**
    Copy `.env.example` to `.env.local` and add your keys.
    ```bash
    cp .env.example .env.local
    ```
    *You'll need a Supabase project and a Google Gemini API key.*

4.  **Run it**
    ```bash
    npm run dev
    ```
